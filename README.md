# Personal Site (GitHub Pages)

This repository is intended to be published via **GitHub Pages** at:

- https://abhisheksharma026.github.io

## Local development (optional)

If you want to preview locally:

```bash
bundle install
bundle exec jekyll serve
```

Then open: http://localhost:4000

## Content structure

- Blog posts: `_posts/` (chronological engineering notes)
- Research notes: `_research/` (paper reviews, critiques)
- Projects: `_projects/` (portfolio entries with reproducibility links)
